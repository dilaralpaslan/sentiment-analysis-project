# Sentiment Analysis Project

This project is a machine learning based sentiment analysis system developed using Python and scikit-learn.  
It classifies YouTube comments as positive, negative, or neutral.

## Features

- Loads comment data from a CSV file
- Cleans and prepares text data
- Uses TF-IDF vectorization
- Trains a Logistic Regression model
- Evaluates model performance
- Predicts sentiment for custom comments

## Technologies Used

- Python
- pandas
- scikit-learn
- TF-IDF
- Logistic Regression

## Dataset

The dataset contains YouTube comments with sentiment labels:

- positive
- negative
- neutral
